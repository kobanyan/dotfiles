# dotfiles

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/kobanyan/dotfiles/master/bin/install.sh | bash
wget -qO https://raw.githubusercontent.com/kobanyan/dotfiles/master/bin/install.sh | bash
```

## Test

```bash
bundle install
bundle exec rspec
```

## Develop

```bash
bundle install
bundle exec guard
```
