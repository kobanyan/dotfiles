# dotfiles

## Install

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/kobanyan/dotfiles/master/bin/install.sh)"
bash -c "$(wget -qO - https://raw.githubusercontent.com/kobanyan/dotfiles/master/bin/install.sh)"
```

## Test

```bash
bundle install
bundle exec rspec
```

## Develop

```bash
bundle install
bundle exec guard
```
